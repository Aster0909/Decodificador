Decodificador de texto 

Usando as seguintes tecnologias:
-HTML: Utilizado para estruturar o conteúdo da página.
-CSS: Utilizado para estilizar a interface do usuário.
-JavaScript: Utilizado para adicionar interatividade à página.

Projeto criado para Encriptar e Desencriptar textos.
Responsivo garantindo uma experiencia de uso otimizado, tanto em dispositivos móveis quanto em desktops.
Ele possue um interface simples com tres botões interativos:

1-Encriptar: Converte o texto inserido em um formato encriptado.
2=Desencriptar: Converte o texto encriptado de volta ao formato original.
3-Copiar: Copia o texto processado para a área de transferência.
